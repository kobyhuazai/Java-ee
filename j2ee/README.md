## 一个基本的CRUD项目

### 功能介绍

英雄的信息有：id，姓名，血量，伤害。

其中，姓名，血量，伤害是添加和编辑时需要输入的信息，id由数据库自动生成，并且能够返回赋予英雄，在英雄的清单页面，每一条都具有edit，delete的操作按键，如下所示：

| id   | name  | hp   | damage | edit     | delete     |
| ---- | ----- | ---- | ------ | -------- | ---------- |
| 1    | Jay   | 90   | 20     | [edit]() | [delete]() |
| 2    | Jones | 88   | 30     | [edit]() | [delete]() |
